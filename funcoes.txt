REQUISITOS TÉCNICOS (
  O sistema deve ser totalmente funcional, atendendo aos seguintes critérios:

Lógica e Estrutura de Dados

- Não utilizar variáveis mutáveis ou efeitos colaterais, exceto onde for estritamente necessário para entrada e saída (I/O).
- Utilizar tipos algébricos para representar:
  - Livros
  - Usuários
  - Empréstimos
- Utilizar funções puras para manipulação dos dados.
- Utilizar listas e funções de ordem superior para processar coleções.

Persistência de Dados

- Persistir os dados em arquivos de texto simples, com suporte a:
  - Leitura
  - Escrita

Interface do Usuário

- Interface via linha de comando, com:
  - Menu textual principal
  - Submenus aninhados
  - Possibilidade de retorno entre menus

Implementações com Recursão Direta

Os seguintes requisitos devem ser implementados usando recursão direta, sem o uso de funções de ordem superior:

- Busca binária por código único.
- Verificação recursiva de duplicatas em múltiplas listas de espera.
